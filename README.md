# Hoang Huynh - Personal Portfolio

🔗 **Live Demo:** [hoanghuynh-dev.vercel.app](https://hoanghuynh-dev.vercel.app)

## ✨ Features

- **Modern UI/UX:** Designed with a clean and interactive interface.
- **Responsive Design:** Fully responsive across all devices (mobile, tablet, desktop).
- **Interactive Animations:** Uses particle backgrounds, typewriter effects, and 3D tilt hover effects to engage users.
- **Smooth Navigation:** Seamless scrolling between different sections (About, Projects, Resume, Contact).
- **GitHub Integration:** Displays live GitHub contribution calendar.

## 🛠️ Technologies & Tools

This project is built using the following stack:

**Core**
- [React 19](https://react.dev/) - UI Library
- [TypeScript](https://www.typescriptlang.org/) - Static Typing
- [Vite](https://vitejs.dev/) - Frontend Build Tool & Bundler

**Styling & UI Components**
- [Bootstrap 5](https://getbootstrap.com/) & [React-Bootstrap](https://react-bootstrap.github.io/) - Layout and responsive UI components
- [React Icons](https://react-icons.github.io/react-icons/) - SVG Icon library
